# CC1110/CC1111 Reference

## Official Documentation

**Datasheet**: [CC1110Fx/CC1111Fx Low-Power SoC with MCU, Sub-1 GHz RF Transceiver, and USB Controller](https://www.ti.com/lit/ds/symlink/cc1111.pdf)

Texas Instruments Part Numbers: CC1110Fx, CC1111Fx

## Quick Reference

### Device Specifications

- **MCU**: Enhanced 8051, 24/26 MHz crystal
- **Flash**: Up to 32 KB programmable
- **RAM**: Up to 4 KB
- **Frequency Bands**: 
  - 300-348 MHz
  - 391-464 MHz  
  - 782-928 MHz
- **Data Rates**: Up to 500 kBaud
- **TX Power**: Up to +10 dBm
- **RX Sensitivity**: -110 dBm @ 1.2 kBaud
- **USB**: Full-speed USB 2.0 (CC1111 only)
- **Interfaces**: UART, SPI, I2C, ADC, Timers, DMA

### Key Features

- Hardware AES-128 encryption/decryption
- Multiple modulation formats: 2-FSK, GFSK, MSK, OOK/ASK, 4-FSK
- Programmable channel bandwidth
- Forward Error Correction (FEC)
- Manchester encoding support
- Data whitening
- Automatic frequency offset compensation
- Link Quality Indicator (LQI)
- Digital RSSI

### Register Overview

**Radio Configuration Registers (0xDF00-0xDF3D)**:
- SYNC1/0: Sync word (0xDF00-01)
- FREQ2/1/0: Frequency control (0xDF09-0B)
- MDMCFG4-0: Modem configuration (0xDF0C-10)
- PKTCTRL1/0: Packet control (0xDF03-04)
- PA_TABLE0-7: Power amplifier settings (0xDF2E-27)
- RSSI, LQI, MARCSTATE: Status registers (0xDF3A-3B)

**Command Strobes (X_RFST = 0xE1)**:
- 0x00: SFSTXON (Frequency synth on)
- 0x01: SCAL (Calibrate)
- 0x02: SRX (Enable RX)
- 0x03: STX (Enable TX)
- 0x04: SIDLE (Exit RX/TX, idle)

### Configuration Formulas

**Frequency**:
```
FREQ_REG = (f_carrier * 2^16) / f_XOSC
```

**Data Rate**:
```
R_DATA = (256 + DRATE_M) * 2^DRATE_E * f_XOSC / 2^28
```

**Channel Bandwidth**:
```
BW_CHANNEL = f_XOSC / (8 * (4 + CHANBW_M) * 2^CHANBW_E)
```

**Deviation** (FSK):
```
DEVIATION = (8 + DEVIATION_M) * 2^DEVIATION_E * f_XOSC / 2^17
```

Where f_XOSC = crystal frequency (typically 24 or 26 MHz)

## Important Notes

### Critical Requirements
1. Radio must be in IDLE state before modifying configuration registers
2. Frequency-dependent settings: VCO selection, power levels, front-end configuration
3. USB 3.0 ports may cause issues; prefer USB 2.0

### Common Use Cases

**433 MHz ASK/OOK** (Key Fobs):
- MOD_ASK_OOK, no sync word
- 1-10 kBaud
- Fixed packet length
- Maximum power

**915 MHz GFSK** (Wireless Sensors):
- MOD_GFSK, 16/16 sync
- 38.4-250 kBaud
- Variable packet with CRC
- Optional FEC

**868 MHz 2FSK+Manchester** (European ISM):
- MOD_2FSK with Manchester encoding
- 4.8-19.2 kBaud
- Sync word enabled
