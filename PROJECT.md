# GoCat

This is a golang project that will mimic the functionality of the famous python program "rfcat."

## Requirements

1. must primarily run on Linux
2. must be in go with no non-go Requirements
3. must have a library useful for building other go programs
4. does NOT require the repl command line use pattern of rfcat
5. must have the ability to easily write simple go programs as easily as using a repl python env to experiment
6. must support ability to dump rf data to disk for separate analysis.

## Testing

Will use two YardStick One devices plugged into the same computer and will send messages to each other.
